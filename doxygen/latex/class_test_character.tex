\doxysubsection{Detailed Description}
Test class for \doxyref{Character}{p.}{class_character}, utilizing the Cpp\+Unit framework. 

This class is designed to implement test cases for testing the functionality of the \doxyref{Character}{p.}{class_character} class. It aims to verify the correctness of the character initialization, ability score generation, ability modifiers calculation, hit points calculation, armor class calculation, attack bonus calculation, damage bonus calculation, and the equipment functionality.
